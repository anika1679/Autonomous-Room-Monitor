# Autonomous-Room-Monitor

To run the navigation interface that saves the map at the end of the run use the command below in the Autonomous-Room-Monitor directory.
```
python src/room_monitor_robot/export.py
```
*Right now you have to manually map out the space with the keyboard and once done Ctrl^C in the terminal to save the map.*

To access the straightened out maps and raw maps, go to ```src/room_monitor_robot/maps```. It will have two different directories raw and corrected each with the respective files.
